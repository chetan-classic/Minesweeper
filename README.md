## Minesweeper

### **Description**
In this game, I have connected the websocket for the game logic. I have use these technologies for build this game :
 - ReactJs
 - Redux
 - Redux-saga
 - Material-UI
 - Ant Designs
 - Type Script

## Installation

```bash
$ npm install
```

## Running the app

```bash
# To Run Game
$ npm start

# To Test
$ npm test

# To Create Build
$ npm run build
```
